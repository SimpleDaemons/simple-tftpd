Welcome to Simple TFTP Daemon
==============================

Thank you for installing Simple TFTP Daemon v0.2.1!

Simple TFTP Daemon is a lightweight, cross-platform TFTP server that provides a secure and efficient way to transfer files using the Trivial File Transfer Protocol.

What's Included
---------------

This package includes:
- The simple-tftpd executable
- Configuration files and examples
- Documentation in the docs/ directory
- Service files for system integration

Quick Start
-----------

1. Review the documentation in the docs/ directory
2. Copy the example configuration from config/simple-tftpd.conf.example
3. Edit the configuration to match your environment
4. Start the service using the provided service files

For detailed installation and configuration instructions, please refer to the documentation in the docs/ directory.
